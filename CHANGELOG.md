# RingMenu Reborn Changelog

## [1.0.0] - 2025-12-19

Complete rewrite of Ring Menu from the ground up. This is a new addon that achieves feature parity with the original while being fully compatible with the current Classic Era client.

### Features
- Radial action wheel that appears at cursor position
- Customizable keybindings per wheel
- Multiple wheel support with independent configurations
- Quick Cast mode: hold keybind to show wheel, release to activate hovered slot
- Configurable button count (1-24 buttons per wheel)
- Adjustable wheel radius and rotation offset
- Background tint color picker with alpha
- Dismiss on use toggle
- Multiple wheels open simultaneously option
- Masque skin support
- Settings accessible via `/rmr` or `/radialwheel`

### Technical
- Built for WoW Classic Era (Interface 11506)
- Uses SecureActionButtonTemplate for combat-safe actions
- Per-character saved variables
